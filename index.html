Welcome to the Cloudfits Official API

This API provides access to various functionalities of the Cloudfits platform.

Endpoints

  • Products

    Retrieve a list of products available on the Cloudfits platform.

    Example Request: GET /products

    Query Parameters Available:

    • count : number
    • sort : 'asc' | 'desc'
    • category : categories

  • Categories

    Retrieve a list of product categories.

    Example Request: GET /categories
